System Capabilities

Designed for Precision & Scale

Multi-Order N-Grams

Dynamically switches and combines models from 1-gram to 6-gram levels, capturing deep contexts and local phrasing with unparalleled responsiveness.

Laplace K-Smoothing

Sophisticated mathematical smoothing handles out-of-vocabulary terms gracefully, ensuring predictions remain continuous and structurally sound.

Sub-Millisecond Inference

Engineered for lightweight, lightning-fast edge inference. Ideal for smart composers, IDEs, and heavy-use professional editors.

import pickle
from src.part_b_language_model import count_ngrams

# Build models from training tweets corpus
ngram_models = {}
for n in range(1, 7):
    ngram_counts = count_ngrams(train_sentences, n)
    prev_counts = defaultdict(int)
    for ngram in ngram_counts:
        prev_counts[ngram[:-1]] += ngram_counts[ngram]
    ngram_models[n] = (ngram_counts, dict(prev_counts))

# Save model parameters as binary artifacts
with open('data/processed/ngram_models.pkl', 'wb') as f:
    pickle.dump(ngram_models, f)
from src.part_d_autocomplete import get_suggestions

# Load model from saved files
ngram_models = pickle.load(open('data/processed/ngram_models.pkl', 'rb'))
vocabulary = pickle.load(open('data/processed/vocabulary.pkl', 'rb'))

# Retrieve predictions dynamically for real-time text input
input_phrase = "how are you"
suggestions = get_suggestions(input_phrase, ngram_models, vocabulary, k=1.0, top_k=5)

print(f"Input: {input_phrase} -> Suggestions: {suggestions}")
# Outputs -> Suggestions: ['today', 'feeling', 'python', 'am', 'intelligence']
